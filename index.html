Помощь по использованию

Как использовать платежный календарь

  • Нажмите на любой день в календаре, чтобы просмотреть или добавить операции
  • Используйте форму добавления операции для ввода доходов и расходов
  • В модальном окне дня можно быстро добавить операцию без закрытия окна
  • Создавайте повторяющиеся операции для регулярных платежей
  • Экспортируйте данные для резервного копирования или переноса на другой компьютер
  • Импортируйте данные для восстановления из резервной копии

Финансовая аналитика

  • Сводка за месяц показывает финансовые показатели только за текущий месяц
  • Аналитика по неделям позволяет отслеживать динамику доходов и расходов
  • Средние показатели в день помогают планировать ежедневный бюджет
  • График денежного потока показывает накопленный баланс по дням
  • Отрицательный баланс на графике выделяется красным цветом
  • Система автоматически определяет возможные кассовые разрывы
